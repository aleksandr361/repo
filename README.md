# Моя игра
Название: космический собиратель
описание: Вас встречает меню с кнопками ИГРАТЬ и МОИ РЕЗУЛЬТАТЫ, при нажатии ИГРАТЬ появляется 
летающий корабль, котторый может двигаться во все стороны. в это время в его сторону 
летят астероиды, столкновение с которыми приведет к смерти. корабль летает и собирает 
разные предметы, набрав определенное количество этих предметов игра завершается. Когда игра завершилась,
высвечивается таблица с результатами, результаты заносятся в базу данных, которую можно посмотреть 
по кнопке МОИ РЕЗУЛЬТАТЫ. вид сверху. Из физики присутствует инерция. программа написана с
использованием библиотеки arcade.
